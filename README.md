# CQU Thesis Editor

这是 CQU Thesis Editor 的公开下载页。软件面向重庆大学普通本科毕业论文（设计）写作，提供 Markdown 编辑、实时预览和一键导出 Word 的桌面工具。

普通使用者不需要下载源码，只需要在 **Releases** 页面下载最新版压缩包。

## 使用前提醒

- 本软件适用于 Windows 10/11。
- 普通使用者只需下载压缩包，解压后双击 `CQUThesisEditor.exe`。
- 导出 Word / 预览 PDF 需要本机安装 Microsoft Word。
- 第一次运行会在程序目录附近生成配置文件。
- 中文封面和英文封面会自动生成，但题目、姓名、学号、专业、导师、学院、日期等个人信息必须自行核对。
- 最终提交论文前，请人工核对中英文封面、目录、页眉页码、公式、图表、参考文献和学院个别要求。
- 本页为便于使用者下载和了解软件而设置，不公开整个源码仓库。

## 主要功能

- Markdown 写作与实时预览。
- 中文封面和英文封面导出，并提醒核对个人信息。
- Word 导出，自动处理页眉、页码、目录、正文样式、参考文献和公式。
- 单面 / 双面打印模式；双面模式下自动在中文封面和英文封面后插入空白背页。
- 行间公式导出为 Word 可编辑 OMML，居中排版，编号右对齐。
- 表格、图片、图题、表题和注释导出。
- 参考文献顺序编码制检查、GB/T 7714 常见类型检查、标题引用误用检查。
- 导出后“格式体检”按钮，可检查常见机器可识别问题。

## 重庆大学本科毕业论文格式要求汇总

以下内容根据项目内置的重庆大学普通本科毕业论文（设计）2023 年修订版规范、学校 Word 模板和编辑器格式规则整理。实际提交前仍应以学校、学院和指导教师的最新要求为准。

### 1. 页面设置

| 项目 | 要求 |
|---|---|
| 纸张 | A4，21cm × 29.7cm |
| 上边距 | 3.0cm |
| 下边距 | 2.5cm |
| 左边距 | 2.5cm |
| 右边距 | 2.5cm |
| 装订线 | 1.0cm |
| 页眉距离 | 1.6cm |
| 页脚距离 | 1.5cm |

### 2. 正文与标题

| 内容 | 字体与字号 | 段落要求 |
|---|---|---|
| 正文 | 中文宋体，西文 Times New Roman，12pt（小四） | 两端对齐，首行缩进约 2 字符，固定 20 磅行距，段前段后 0 |
| 一级标题 | 中文黑体，西文 Times New Roman，16pt（三号），加粗 | 居中，段前 20pt，段后 20pt，固定 20 磅行距 |
| 二级标题 | 中文黑体，西文 Times New Roman，15pt（小三），加粗 | 居左，段前 20pt，段后 0，固定 20 磅行距 |
| 三级标题 | 中文黑体，西文 Times New Roman，14pt（四号），加粗 | 居左，段前段后 0，固定 20 磅行距 |
| 表题、图题 | 中文宋体，西文 Times New Roman，10.5pt（五号） | 居中 |
| 图注、表注 | 五号基准 | 位于图表附近，导出后需人工复核分页位置 |

特殊一级标题建议按模板显示：

- 摘要：`摘　要`
- 目录：`目　录`
- 参考文献：`参 考 文 献`
- 致谢：`致　谢`
- 附录：`附　录`

### 3. 封面与前置部分

- 中文封面和英文封面不显示页眉、页脚和页码。
- 中文封面和英文封面均按单面封面处理。
- 若选择双面打印模式，软件会在中文封面后、英文封面后自动插入空白背页。
- 中文摘要、英文摘要和目录属于前置部分，页码通常使用小写罗马数字。
- 正文从第一章开始使用阿拉伯数字页码，并从 1 起算。

### 4. 页眉与页码

| 部分 | 页眉 | 页码 |
|---|---|---|
| 封面 | 无 | 无 |
| 摘要、目录等前置部分 | 左侧为“重庆大学本科学生毕业论文（设计）”，右侧为当前章节标题 | 小写罗马数字，页脚居中 |
| 正文 | 左侧为“重庆大学本科学生毕业论文（设计）”，右侧为当前章节标题 | 阿拉伯数字，页脚居中，从 1 开始 |

页眉文字建议使用宋体 / Times New Roman，9pt（小五），页眉下方为黑色单线边框。页码建议使用宋体 9pt（小五），位于页脚居中。

双面打印时，偶数页和奇数页页眉规则会有所区分：偶数页偏向学校标题，奇数页偏向当前章节标题。正文超过 30 页时，建议优先考虑双面打印并人工翻页核对。

### 5. 表格

- 表题位于表格上方，格式建议为 `表 章号.序号 标题`。
- 表题字体为五号，居中。
- 表格正文和表头一般使用宋体 10.5pt（五号）。
- 表头可加粗，表格内容水平和垂直居中。
- 常规表格按三线表处理：表格上边框和下边框较粗，通常不使用密集竖线。
- 若存在合计行，可在合计行上方增加细线，形成四线表效果。
- 表注位于表格下方，导出后需检查是否和表格分页分离。

### 6. 图片

- 图片居中。
- 图题位于图片下方，格式建议为 `图 章号.序号 图题`。
- 图题字体为五号，居中。
- 图片宽度应适应正文版心，不应超出页边距。
- 图注位于图题或图片附近，导出后需检查清晰度、比例和分页位置。

### 7. 公式

- 行间公式居中。
- 公式编号右对齐，格式为 `(章节号.公式序号)`，例如 `(1.1)`。
- 公式应尽量转换为 Word 可编辑 OMML 公式，便于后续修改。
- 正文中的短公式应与正文在同一行。
- 复杂矩阵、分段函数、特殊宏或颜色等细节，导出后建议在 Word 公式编辑器中人工复核。

### 8. 目录

- 目录标题建议显示为 `目　录`，居中。
- 目录应自动读取正文标题层级。
- 目录页码右对齐，使用点线前导符。
- 正文修改后，应在 Word 中右键目录并选择“更新整个目录”。

### 9. 参考文献

- 参考文献标题建议显示为 `参 考 文 献`。
- 参考文献条目按顺序编码制排列，如 `[1]`、`[2]`。
- 参考文献内容字号为五号 10.5pt，不使用正文小四字号。
- 参考文献条目建议使用悬挂缩进，固定 20 磅行距。
- 正文引用应以上标形式呈现，常见写法包括 `[1]`、`[1, 3, 5]`、`[1-3]`。
- 参考文献条目编号本身不应被误处理为上标。
- 著录格式参考 GB/T 7714-2015，常见类型包括：
  - 期刊文章 `[J]`
  - 专著 `[M]`
  - 学位论文 `[D]`
  - 会议论文 `[C]`
  - 网页或在线资料 `[EB/OL]`

### 10. 导出后最终核对清单

- 页面是否为 A4，页边距、装订线、页眉页脚距离是否正确。
- 中文封面和英文封面个人信息是否完整、准确。
- 中文封面和英文封面是否无页眉、无页码。
- 双面打印模式下，封面背页是否为空白页。
- 摘要、目录等前置部分页码是否为罗马数字。
- 正文第一页页码是否从 1 开始。
- 页眉右侧章节名是否正确，页眉线是否存在。
- 一级、二级、三级标题字体、字号和对齐方式是否正确。
- 正文是否为小四、固定 20 磅行距、首行缩进约 2 字符。
- 图题是否在图下方，表题是否在表上方。
- 公式是否居中，编号是否右对齐。
- 目录是否已更新到最新页码。
- 正文引用和参考文献条目是否一一对应。
- 参考文献条目字号是否为五号 10.5pt。
