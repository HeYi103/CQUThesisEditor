# CQU Thesis Editor

这是 CQU Thesis Editor 的公开下载页。

## 下载方式

请在右侧或下方的 **Releases** 页面下载最新版压缩包：

- `CQUThesisEditor-share-20260628.zip`

压缩包内包含：

- `CQUThesisEditor.exe`
- `使用说明.md`

## 使用前提醒

- 本软件适用于 Windows。
- 普通使用者只需下载压缩包，解压后双击 `CQUThesisEditor.exe`。
- 导出 Word / PDF 需要本机安装 Microsoft Word。
- 第一次运行会在程序目录附近生成配置文件。
- 最终提交论文前，请人工核对中英文封面、目录、页眉页码、公式、图表和参考文献。

## 致谢与来源说明

本软件的格式规范、论文结构和模板理解，深受 [CQUThesis](https://github.com/nanmu42/CQUThesis) 项目的启发与帮助。CQUThesis 是重庆大学毕业论文 LaTeX 模板，由 Zhennan Li（[@nanmu42](https://github.com/nanmu42)）等前辈同学长期建设和维护，为重庆大学同学提供了高质量的论文排版参考。

在此特别向 CQUThesis 项目作者 Zhennan Li、重庆大学 TeX 用户组，以及历任参与 CQUThesis 模板维护、测试、反馈和格式规范整理的前任团队同学致以真挚感谢。正是他们此前对论文格式、模板结构、参考文献样式和使用文档的持续打磨，才让后续的 Word 编辑器封装、格式自动化和使用体验完善有了可靠基础。

本项目公开下载页只提供可执行软件和使用说明；如有格式细节出入，仍请以学校、学院和指导教师的最新要求为准。
