# CQU Thesis Editor

这是 CQU Thesis Editor 的公开下载页。

## 下载方式

请在右侧或下方的 **Releases** 页面下载最新版压缩包：

- `CQUThesisEditor-share-20260628.zip`

压缩包内包含：

- `CQUThesisEditor.exe`
- `使用说明.md`

## 使用前提醒

- 本软件适用于 Windows。
- 普通使用者只需下载压缩包，解压后双击 `CQUThesisEditor.exe`。
- 导出 Word / PDF 需要本机安装 Microsoft Word。
- 第一次运行会在程序目录附近生成配置文件。
- 最终提交论文前，请人工核对中英文封面、目录、页眉页码、公式、图表和参考文献。
