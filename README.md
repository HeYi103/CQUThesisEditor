# CQU Thesis Editor

重庆大学本科毕业论文（设计）桌面编辑器。面向普通使用者的下载方式是进入 GitHub Releases，下载最新的分享压缩包。

## 下载与使用

1. 打开 Releases 页面。
2. 下载最新版本的 `CQUThesisEditor-*.zip`。
3. 解压后双击 `CQUThesisEditor.exe`。
4. 阅读压缩包内的 `使用说明.md`。

压缩包通常只包含：

- `CQUThesisEditor.exe`
- `使用说明.md`

不需要下载或公开使用源码即可运行软件。

## 主要功能

- Markdown 写作与实时预览。
- 中文封面和英文封面导出，并提醒核对个人信息。
- Word 导出，自动处理页眉、页码、目录、正文样式、参考文献和公式。
- 单面 / 双面打印模式；双面模式下自动在中英文封面后插入空白背页。
- 行间公式导出为 Word 可编辑 OMML，居中排版，编号右对齐。
- 表格、图片、图题、表题和注释导出。
- 参考文献顺序编码制检查、GB/T 7714 常见类型检查、标题引用误用检查。
- 导出后“格式体检”按钮，可检查常见机器可识别问题。

## 系统要求

- Windows 10/11。
- Microsoft Word。导出 Word 后处理和 PDF 预览依赖 Word COM。
